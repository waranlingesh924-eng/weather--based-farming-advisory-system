# 🌾 Weather Based Farming Advisor System

A complete **8-module farming advisory website** built with pure **HTML, CSS & JavaScript**.  
Designed for college final-year projects. Works on **GitHub Pages** with zero backend.

---

## 📌 Project Overview

This system guides farmers through 8 intelligent steps to get complete crop guidance:

| Module | Purpose |
|--------|---------|
| 1 – Location & Climate | Collect state, district, village & climate |
| 2 – Crop & Soil | Select crop, soil type & planting month |
| 3 – Suitability Analysis | Check crop–soil–climate compatibility |
| 4 – Disease Prediction | Predict diseases with causes & prevention |
| 5 – Fertilizer Recommendation | Suggest fertilizer, quantity & timing |
| 6 – Irrigation Advisory | Recommend irrigation method & schedule |
| 7 – Market Price | Show current price & trend from APMC data |
| 8 – Final Report | Complete advisory summary + print option |

---

## 🌐 How to Deploy on GitHub Pages

1. **Create a new GitHub repository**  
   Go to [github.com](https://github.com) → New Repository → Name it (e.g., `farming-advisor`)

2. **Upload the files**  
   Upload all 4 files:
   - `index.html`
   - `style.css`
   - `app.js`
   - `README.md`

3. **Enable GitHub Pages**  
   - Go to your repository → **Settings** → **Pages**
   - Under **Source**, select **Deploy from a branch**
   - Choose **main** branch → **/ (root)** → Click **Save**

4. **Access your live site**  
   Your website will be live at:  
   `https://your-username.github.io/farming-advisor/`

- ✅ **15 Indian states** with real districts
- ✅ **20+ crops** covered (Rice, Wheat, Cotton, Tomato, etc.)
